Haml Coffee Online

Hi! I'm Haml-Coffee, a JavaScript templating solution that implements Haml with CoffeeScript for its logic. I run on the server side on Node.js and also in the Browser. You can use me with Haml Coffee Assets also with Ruby on Rails or any framework that supports Sprockets like Sinatra.
